# CS3219 Project (PeerPrep) - AY2526S2
## Group: G11

## Docker Compose

1. Add these files:
   - `user-service/.env`
   - `question-service/.env`
   - `ai-assistant-service/.env`
   - `matching-service/application.properties`

2. Build:

```bash
docker compose build
```

3. Start:

```bash
docker compose up -d
```

4. Open:

```text
http://localhost:3001
```

5. Stop:

```bash
docker compose down
```
